# Meteor Shine

Open Source Meteor CMS Project

## Features

* Custom accounts-ui
* CRUD: Post with category, and comment
* Markdown editor
* Image upload to [cloudinary.com](http://cloudinary.com/) cloud service
* I18n ready
* Notification
* Separated Admin app
* Deploy to your own server using Nginx + Phusion Passenger
